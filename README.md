# 依赖
  1. element-plus
  ```shell
  # NPM
  $ npm install element-plus --save

  # Yarn
  $ yarn add element-plus

  # pnpm
  $ pnpm install element-plus
  ```
  2. nprogress
  ```shell
    npm install nprogress
    npm install @types/nprogress -D
  ```
  3. pinia
  ```shell
    npm install pinia
  ```
  4. axios
  ```shell
    npm install axios
  ```
  5. lodash
  ```shell
    npm install --save-dev @types/lodash
  ```
  6. mitt
  ```shell
    npm install mitt -S
  ```
